# Política de Seguridad de Gungo

Gungo (Farándula y Noticias Latam) toma muy en serio la seguridad de su plataforma y de sus visitantes. Esta política describe cómo gestionamos las vulnerabilidades, qué versiones mantenemos y cómo puedes ayudarnos reportando problemas de manera segura.

## Versiones Soportadas

Actualmente, solo las siguientes versiones de nuestro proyecto reciben actualizaciones y parches de seguridad críticos.

| Versión | Soporte de Seguridad | Notas |
| :--- | :--- | :--- |
| **2.x** | ✅ Activo | Versión actual y principal (Recomendada). |
| **1.x** | ✅ Activo | Mantenimiento crítico de seguridad hasta junio de 2026. |
| **0.x** | ❌ No soportado | Actualiza inmediatamente a la versión más reciente. |
| **< 1.0** | ❌ No soportado | Fuera de soporte oficial. |

*Nota: Solo las versiones marcadas con ✅ reciben parches de seguridad urgentes.*

## Cómo reportar una vulnerabilidad

**Por favor, NO abras un Issue público para reportar vulnerabilidades de seguridad.** Hacerlo pondría en riesgo la plataforma antes de que podamos aplicar una solución.

Para garantizar la seguridad de nuestra infraestructura y de los lectores de Gungo, te pedimos que sigas este proceso de divulgación coordinada:

1. **Reporte Privado:** Utiliza la función **"Private vulnerability reporting"** directamente en la pestaña "Security" de este repositorio en GitHub para enviarnos un aviso confidencial.
2. **Alternativa por Correo:** Si no tienes cuenta de GitHub, envía un correo electrónico detallado a `seguridad@gungo.com` (reemplaza con tu correo real).
3. **Información Requerida:** Incluye una descripción clara de la vulnerabilidad, los pasos para reproducirla y el posible impacto (ej. XSS, Bypass de Rate Limiting).
4. **Tiempo de Respuesta:** Nuestro equipo de desarrollo confirmará la recepción de tu reporte en un plazo máximo de 48 horas y te mantendrá informado sobre el progreso del parche.

Agradecemos profundamente a la comunidad de investigadores de seguridad que nos ayudan a mantener a Gungo como una plataforma robusta y blindada.
